# todo-app

## 概要
シンプルなToDo管理アプリです。

## 使い方
```
このToDoアプリの使い方は以下の通りです。

1. アプリを起動すると「コマンドを入力してください(new, List, delete, quit)」と表示されます。
2. 入力できるコマンドは次の4つです。

- `List`  
  現在のToDoリストを一覧表示します。各ToDoには番号（インデックス）が振られています。

- `new`  
  新しいToDoを追加します。コマンド入力後、「新しいTodoを入力してください」と表示されるので、追加したい内容を入力してください。

- `delete`  
  ToDoを削除します。「削除するインデックスを入力してください」と表示されるので、一覧表示で確認した番号を入力してください。

- `quit` または `q`  
  アプリを終了します。

コマンドを入力し、必要に応じて追加・削除を繰り返すことができます。終了したい場合は `quit` または `q` を入力してください。
